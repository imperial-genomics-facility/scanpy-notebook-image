# Notebook container image for running single cell data analysis using scanpy
A repository for keeping conda env and docker image file for running single cell analysis using scanpy

## Installed packages

 * Python v3.6.9
 * Scanpy v1.4
 * R v3.5.1
 * Scran v1.10.1
 * Louvain v0.6.1
